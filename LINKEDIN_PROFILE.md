### Project 1: AI Resume Parser (SaaS Application)
**Role:** Full Stack Developer
**Technologies:** Next.js 15, Google Gemini AI, Express.js, MongoDB
*   Built and deployed an AI-powered resume optimization tool that parses complex document formats (PDF, DOCX) and provides real-time ATS scoring.
*   Integrated Google Gemini 2.0 Flash API to generate intelligent suggestions for tone adjustment and actionable resume improvements.
*   Engineered a robust backend with Express.js handling file processing, validation, and error management with 99.9% uptime on Render.

### Project 2: StockPilot (Inventory Management System)
**Role:** Lead Developer
**Technologies:** Next.js, MongoDB, Docker, JWT Auth
*   Architected a production-grade inventory management system for SMBs, featuring real-time stock tracking and low-inventory alerts.
*   Implemented a custom "Offline Mode" architecture that allows the app to function without internet connectivity, syncing data automatically when online.
*   Designed complex MongoDB aggregations to visualize stock movement trends and analytics for 1000+ SKUs.
*   **Live Demo:** stock-pilot-wheat.vercel.app

### Project 3: TenantVerse (Property Management Platform)
**Role:** Full Stack Developer
**Technologies:** Next.js, Stripe API, Role-Based Access Control (RBAC)
*   Developed a comprehensive SaaS platform for property managers to handle multiple tenants, leases, and maintenance requests in one dashboard.
*   Integrated Stripe Payment Intent API to securely process rent payments and generate automated invoices having PCI compliance.
*   Implemented strict Role-Based Access Control (RBAC) to secure sensitive data between Admin, Manager, and Tenant user roles.
